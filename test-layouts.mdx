## Slide 2 - Layout Horizontal 

Este slide usa o layout horizontal. O texto deve aparecer à esquerda e as imagens à direita.

**Características do layout horizontal:**

- Texto fica na coluna da esquerda
- Imagens ficam na coluna da direita
- Layout responsivo para diferentes tamanhos de tela
- Melhor aproveitamento do espaço horizontal

## Slide 4 - Texto Longo com Layout Horizontal

Demonstração de como o layout horizontal funciona com mais texto.

**Sistema SCADA - Componentes Principais:**

1. **Sensores e Atuadores**: Dispositivos de campo que coletam dados e executam comandos
2. **Unidades Remotas (RTUs)**: Concentram sinais de sensores e controlam atuadores
3. **Sistema de Comunicação**: Redes que transportam dados entre campo e centro de controle
4. **Centro de Controle**: Interface humano-máquina para supervisão e controle
5. **Banco de Dados**: Armazenamento histórico de dados para análise

O sistema permite monitoramento em tempo real, controle remoto, geração de alarmes e relatórios históricos.

## Slide 5 - Voltando ao Layout Vertical

Este slide volta para o layout vertical tradicional.

O comando `---sldlayout1` retorna ao layout padrão onde o texto e as imagens são renderizados sequencialmente na vertical.

**Quando usar cada layout:**
- **Layout 1 (Vertical)**: Ideal para slides com foco no texto, apresentações lineares
- **Layout 2 (Horizontal)**: Ideal para slides com imagens de apoio, comparações visuais
